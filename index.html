Project Overview

This project demonstrates a robust data processing pipeline for sales data. It utilizes Python with the Pandas library for data manipulation, ensures code quality with Ruff, and automates the entire process including deployment with GitHub Actions. The goal is to transform raw Excel data (data.xlsx) into a structured CSV (data.csv), process it using a Python script (execute.py), and publish the resulting summary (result.json) to GitHub Pages.

Source Data

data.xlsx (Excel) → data.csv (CSV)

Processing Logic

Python execute.py with Pandas

GitHub Actions & Ruff

Output & Deployment

result.json on GitHub Pages

How It Works

The project follows a standard CI/CD workflow:

  1. A push to the main branch triggers the GitHub Actions workflow.
  2. The workflow sets up the Python environment (3.11+) and installs necessary libraries (Pandas 2.3, Ruff).
  3. Ruff performs static analysis on execute.py to ensure code quality.
  4. The execute.py script is executed, which reads data.csv (committed) and processes the sales data.
  5. A summary `result.json` is generated by execute.py.
  6. Finally, this `result.json` is published to GitHub Pages, making the processed data publicly accessible.

For detailed technical information, including the full source code for `execute.py` and the `ci.yml` workflow, please refer to the README.md file.
